Student Career Prediction using Machine Learning

Project Overview

This project focuses on predicting student career paths using Machine Learning techniques.
It analyzes student data and builds a predictive model to classify or suggest suitable career outcomes.

 Objectives

* To analyze student data
* To preprocess and clean the dataset
* To build a prediction model using Machine Learning
* To evaluate model performance using metrics

Technologies Used

* Python
* Pandas
* NumPy
* TensorFlow / Keras
* Scikit-learn
* Matplotlib

Project Files

* `student_prediction_microproject.ipynb` → Main implementation notebook
* `Dataset Project 404.xlsx` → Dataset used
* `README.md` → Project documentation

---

 Working Process

1. Data Collection

* Dataset is loaded using Pandas

 2. Data Preprocessing

* Handling missing values
* Converting categorical data
* Feature selection

 3. Model Building

* Neural Network using TensorFlow/Keras
* Layers include Dense and Dropout

4. Model Evaluation

* Accuracy Score
* Precision
* Recall
* F1-Score
* Classification Report


How to Run the Project

1. Open the `.ipynb` file in Jupyter Notebook / Google Colab
2. Upload the dataset file
3. Run all cells step by step
4. View the output and predictions


Output

* Predicts student career category
* Displays evaluation metrics
* Shows performance visualization (if plotted)
